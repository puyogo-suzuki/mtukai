# A temperature and humidity example
This example measures temperature and humidity.  
After some measurement, the main processor wakes up and reports thre results.

## Requirements
 - ESP32-C6
 - SHT3x Temperature and Humidity Sensor

## Connection
### SHT3x Sensor
ESP32-C6 communicates via I2C.
 - SDA - GPIO6
 - SCL - GPIO7
